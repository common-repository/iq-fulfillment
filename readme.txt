== Installation ==

= Minimum Requirements =
* PHP 7.2 or greater is recommended
* Woocommerce plugin

= Automatic installation =
* Log in to your WordPress website and navigate to the "Plugins" section in the sidebar.
* Click on the "Add New" button.
* Search for the "IQ Fulfillment" plugin in the search bar, and click on the "Install Now" button for the plugin you want to add.
* Once the plugin is installed, click on the "Activate" button.
* The plugin should now be activated and added to your website.
* Go to the "IQ Fulfillment" manu from your admin menubar.
* Click for integration & allow the permission.
* Login to our system & wait for the approval.
